# PhysiNet

基于Mamba架构的物理图像重建网络。

## 项目结构

- `src/`: 源代码文件夹
- `data/`: 数据集文件夹
- `experiments/`: 实验结果和配置文件
- `docs/`: 项目文档
- `requirements.txt`: 项目依赖
- `README.md`: 项目说明

## 许可证

本项目采用 MIT 许可证 

## 使用方法

1. 准备数据集
2. 配置训练参数
3. 运行训练
4. 查看结果
5. 使用模型进行推理
